Для использования этой программы, вам потребуется базовое понимание командной строки и некоторых принципов программирования на C++. Программа предназначена для управления данными о футболистах футбольной команды. Вот руководство пользователя:

**1. Добавить игрока:**
   - Эта опция позволяет вам добавить нового футболиста в список.
   - Вы должны ввести Ф.И.О. игрока (латинскими буквами) и дату рождения (в формате дд.мм.гггг).
   - Затем вам потребуется ввести количество игр, количество забитых мячей, количество желтых карточек и количество красных карточек.

**2. Редактировать игрока:**
   - Эта опция позволяет вам редактировать данные о существующем футболисте.
   - Вы должны ввести индекс игрока, которого вы хотите отредактировать, и затем ввести новые данные.

**3. Удалить игрока:**
   - Эта опция позволяет вам удалить футболиста из списка.
   - Вы должны ввести индекс игрока, которого вы хотите удалить.

**4. Вывести список игроков:**
   - Эта опция отобразит список всех футболистов, включая их Ф.И.О., дату рождения, количество игр, количество забитых мячей, количество желтых карточек и количество красных карточек.

**5. Сортировать игроков:**
   - Эта опция позволяет вам отсортировать список футболистов по различным критериям.
   - Выберите поле для сортировки (1-4):
     - 1. Дата рождения
     - 2. Количество игр
     - 3. Количество забитых мячей
     - 4. Количество карточек

**6. Поиск игроков:**
   - Эта опция позволяет вам выполнить поиск футболистов по введенной строке.

**7. Вывести топ-5 бомбардиров:**
   - Эта опция отобразит топ-5 футболистов с наибольшим количеством забитых мячей.

**8. Вывести самого недисциплинированного игрока:**
   - Эта опция определит самого недисциплинированного футболиста команды.

**9. Найти всех игроков, младше 20 лет, забивших более 2 голов:**
   - Эта опция найдет всех футболистов, младше 20 лет, и забивших более 2 голов.

**10. Сохранить данные в текущем файле (CSV формат):**
   - Эта опция позволяет вам сохранить текущий список футболистов в CSV-файл с именем "players.csv".

**11. Сохранить данные в другом файле (CSV формат):**
   - Эта опция позволяет вам сохранить текущий список футболистов в новом CSV-файле. Вам нужно ввести имя нового файла.

**12. Вернуться в главное меню:**
   - Эта опция позволяет вам вернуться в главное меню программы.

**2. О программе:**
   - Эта опция покажет информацию о программе, включая версию и дату последних изменений.

**3. Руководство пользователя:**
   - Эта опция отобразит руководство пользователя. Предполагается, что оно хранится в файле "user_manual.txt" в том же каталоге, где находится программа.

**4. Задание:**
   - Эта опция отобразит описание задания, которое выполняет программа.

**5. Выйти:**
   - Эта опция завершит выполнение программы.

Примечание: Программа автоматически загружает данные из файла "players.csv" при запуске и может сохранить данные в этот же файл при завершении. Пожалуйста, обязательно сохраните данные перед выходом, если вы внесли изменения.
